Gra Arkanoid "Odbijanie piłeczki"

Do stworzenia gry wykorzystano bibliotekę SFML 2.5.1

Opis gry
Celem gry jest pokonanie bossa poprzez zniszczenie kolorowych bloków na planszy za pomocą piłki odbijanej przez paletkę. 
Gra oferuje kilka poziomów trudności.

Główne funkcjonalności:
* System poziomów Kilka zróżnicowanych układów bloków ładowanych dynamicznie.
* Walka z bossem Unikalny przeciwnik (Boss), który porusza się i posiada punkty życia.
* Zapis i odczyt Możliwość zapisania stanu gry do pliku i wczytania go w dowolnym momencie.
* Menu główne Nawigacja między opcjami gry.
* Mechanika odbić Mechanika odbić piłki od ścia, paletki oraz krawędzi ekranu
* System punktów Naliczanie punktów za zbijanie bloków.

Sterowanie:
Strzałka lewo prawo/klawisze "A" i "D" Poruszanie paletką 
F5 zapis gry
F9 Wczytanie gry
ESC Wyjście do menu
